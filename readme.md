# Tsinghua Coordinates

Geographic coordinates of buildings inside campus of Tsinghua University.

清华大学（清华园校区）建筑物地理坐标

* 逗号分隔，每行一个建筑物，首字段为建筑物名称，其后为若干对“经度、纬度”坐标，依次围成多边形。

* 文件中的中文字符均为国标编码。

* 数据截止至 2014 年。各坐标由 Google Earth 读取，经 GPS 作偏移量修正。
